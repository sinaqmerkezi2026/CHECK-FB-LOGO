# **App Name**: Kalkulyator

## Core Features:

- Basic Arithmetic: Perform fundamental addition, subtraction, multiplication, and division operations.
- Number and Decimal Input: Allow users to input digits 0-9 and a decimal point for calculations.
- Operation Buttons: Interactive buttons for arithmetic operators (+, -, ×, ÷) and the equals (=) function.
- Clear Functionality: Provide buttons to clear the last entry (CE) and reset the entire calculation (C).
- Result Display: Show the current input and the calculated result clearly in a dedicated display area.

## Style Guidelines:

- Primary color: A deep, authoritative blue (#2626D9) representing precision and reliability, serving as the main brand accent.
- Background color: A very light, desaturated blue (#F5F5FC) to ensure clarity and visual calm, derived from the primary hue.
- Accent color: A vibrant yet clean sky blue (#5CD5FB) for crucial interaction elements like the equals button, providing clear contrast while maintaining visual harmony.
- Body and headline font: 'Inter' (sans-serif) for its neutral, modern, and highly legible appearance, ideal for numerical and operational display.
- Use standard, minimalist icons for mathematical operations and utility functions to ensure immediate recognition and intuitive interaction.
- Implement a clear, intuitive grid-based layout for calculator buttons, distinctly separating the display from the input pad for optimal usability.
- Apply subtle press-and-release feedback animations to buttons to enhance interactivity and acknowledge user input.
